Reports from adjacent years tend to be the most similar pairs. ## 3. What is Wordfish? **Wordfish** is an unsupervised one-dimensional text scaling method. It uses word frequencies to estimate the position of each document on a latent dimension. In simple terms, it places documents on a line based on how their word choices differ. Documents with similar language are placed close together, and documents with different language are placed farther apart. It is commonly used to scale political texts such as legislative speeches or party manifestos along an ideological or thematic dimension without requiring any prior labeling of documents. ## 3a. Run Wordfish on DOD reports ```{r} #| label: dod-wordfish wf_dod <- textmodel_wordfish(dfmat_dod_trim, dir = c(1, ndoc(dfmat_dod_trim))) summary(wf_dod) ``` ```{r} #| label: dod-wordfish-plot wf_plot_data <- data.frame( document = docnames(dfmat_dod_trim), wordfish_position = as.numeric(wf_dod$theta) ) ggplot(wf_plot_data, aes(x = wordfish_position, y = reorder(document, wordfish_position))) + geom_point(size = 3, color = "#003366") + labs( title = "Wordfish Positions for DOD Reports", x = "Wordfish Position", y = NULL ) + theme_minimal(base_size = 14) + theme( plot.title = element_text(color = "#003366", face = "bold"), axis.text = element_text(color = "#003366"), axis.title = element_text(color = "#003366") ) knitr::kable(wf_plot_data, digits = 3, format = "html") ``` The Wordfish output places the DOD reports along a single dimension. Earlier reports appear at one end of the scale while later reports cluster toward the other end, suggesting that language and strategic framing have shifted over time. The clustering of recent reports indicates their vocabulary has become more standardized. ## 4. How to compare positions One way to validate Wordfish results is to compare them against another scaling method such as **correspondence analysis (CA)**. If both methods place documents in a similar order, that gives more confidence that the pattern is real and not an artifact of one particular algorithm. ```{r} #| label: compare-positions ca_dod <- textmodel_ca(dfmat_dod_trim) dod_positions_compare <- data.frame( document = docnames(dfmat_dod_trim), wordfish = as.numeric(wf_dod$theta), ca_dim1 = as.numeric(coef(ca_dod, doc_dim = 1)$coef_document) ) kable(dod_positions_compare, digits = 3) ``` ```{r} #| label: compare-plot ggplot(dod_positions_compare, aes(x = wordfish, y = ca_dim1, label = document)) + geom_point(size = 3) + geom_text(size = 3, vjust = -0.5, check_overlap = TRUE) + labs( title = "Comparing Document Positions: Wordfish vs Correspondence Analysis", x = "Wordfish Position", y = "Correspondence Analysis Dimension 1" ) ``` ```{r} #| label: compare-correlation cor(dod_positions_compare$wordfish, dod_positions_compare$ca_dim1) ``` **Answer for Question 4:** Positions can be compared by running the same documents through a second scaling method such as correspondence analysis and then computing the correlation between the two sets of scores. A high correlation means both methods detect the same underlying structure, which validates the Wordfish results. If the scores diverge, the dimension being captured may differ between methods. ## 5.
